# GDG Tulsa Website

The official website for **Google Developer Group Tulsa** — a community of developers, students, and builders in Tulsa, Oklahoma. Live at [gdgtulsa.com](https://gdgtulsa.com), deployed via GitHub Pages.

## What's Inside

- **Home (`index.html`)** — Hero with autoplay intro video, animated GDG layout, and Learn / Build / Connect / Grow cards matching the GDG deck
- **About (`about-gdg.html`)** — Team, credentials, communities, and Workspace product info
- **Build with AI 2026 (`bwai.html`)** — Landing page for the Build with AI hackathon, with intro GIFs, save-the-date socials, and an editable signup form
- **Organizers (`organizer.html`)** — Organizer bios and credentials
- **Member Portal (`admin.html`)** — Firebase-backed member portal and admin dashboard (`firebase-config.js`, `firestore.rules`)

## Tech Stack

- Static **HTML / CSS / JavaScript** — no build step required
- **Firebase** (Auth + Firestore) for the member portal and admin dashboard
- **GitHub Pages** for hosting, with a custom domain via `CNAME`
- `globe.js` and `script.js` for UI theming, animation, and interactivity
